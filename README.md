# MyShell scripts

## Installation

```sh
cd ~ && git clone git@github.com:matiascaniete/myshell.git
```

## Configuration and autocompletion zsh

Add this line to your `~/.zshrc`:

```sh
fpath=(~/myshell/autocomplete $fpath)
```
